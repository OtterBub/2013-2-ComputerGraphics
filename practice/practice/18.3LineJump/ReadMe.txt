사용법 2010180020 박성경

--------- 게임 방법 --------

앞에서부터 생성되는 노란 길을 따라 조심히 가면 됩니다.

점프를 한번 한다고 해서 정확히 그 길을 따라 가는 것이 아니므로

점프 도중에 컨트롤을 잘 해야합니다.

자동차 내부의 작은 사각형이 길을 벗어나면 게임 오버,

약 3초뒤 재시작 합니다.

---- 키보드 사용법 ----

j / l : 왼쪽 점프, 오른쪽 점프

점프 중에
j / l : 왼쪽 으로 이동, 오른쪽 으로 이동

ex) 왼쪽 점프시에 오른쪽 이동을 누르면 조금 더 감속해서 이동
왼쪽 점프시에 왼쪽 이동 을 누르면 조금 더 가속해서 이동

i / k : 앞뒤로 이동.

ex) 앞으로 자동 이동시에 뒤로 k를 누르면 속도 감속
앞으로 자동 이동시에 앞으로 i를 누르면 속도 가속


w / s : eye z축 이동
r / f : eye y축 이동


위 / 아래 : center z축 이동
page up / down : center y축 이동


------------- 파일 내용 --------------

main.cpp - main함수가 정의되어 있습니다. 

CallBackFunc.cpp / .h - glut 콜백 함수가 정의 되어있습니다.


State.h - 상태를 정의하기 위한 추상화 객체가 정의 되어있습니다.

MainCharState.cpp / .h - Stage를 상속받아 메인 캐릭터(자동차)의 상태를 정의합니다.


Object3D.cpp / .h - 3D객체를 위한 추상화 객체가 정의 되어있습니다.

Char.cpp / .h - Object3D를 상속받아 MainChar(자동차)를 정의합니다.


World3D.cpp / .h - 3D객체를 관리하기 위한 월드 객체 입니다.

Stage.cpp / .h - World3D를 상속받아 18번 실습의 스테이지를 만들었습니다.


Object.cpp / .h - 2D객체가 정의되어 있습니다.

World.cpp / .h - 2D객체를 관리하기 위한 월드 객체 입니다.
